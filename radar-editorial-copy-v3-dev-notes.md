# Radar Editorial Copy Library Rioplatense V3

## Qué cambia

Esta versión reemplaza la capa anterior de templates con finales reciclados.

Ahora:
- cada frase es cerrada
- no hay remates compartidos
- se elimina la repetición de cierres como muletilla
- se fuerza más variedad de ritmo
- se sube el tono futbolero y la mala leche

## Reglas de uso

1. No repetir una misma frase en la misma jornada y liga.
2. No repetir un mismo patrón de apertura dentro del mismo label.
3. No repetir un mismo patrón de cierre dentro del mismo label.
4. Mezclar contextos, no usar siempre el primero.
5. Mezclar tonos; no llenar toda la jornada con frases venenosas.
6. Si una frase suena a escritorio, se descarta aunque sea válida.

## Rotación sugerida

- cooldown exact phrase: 45 días
- cooldown same opening pattern: 20 días
- cooldown same closing pattern: 20 días
- máximo 1 frase muy venenosa por label en una misma jornada

## Estructura

Cada label tiene:
- `displayName`
- `contexts`
- `tone`
- `templates`

## Nota

El objetivo no es sonar literario ni técnico.
El objetivo es sonar a fútbol rioplatense con calle, picante y fundamento.
